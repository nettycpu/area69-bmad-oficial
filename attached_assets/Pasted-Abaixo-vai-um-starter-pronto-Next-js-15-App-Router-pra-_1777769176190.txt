Abaixo vai um starter pronto (Next.js 15 App Router) pra você criar uma landing page curtinha, mas com visual de “colagem editorial” (papel, polaroid, fita, objetos flutuando), com CTA forte “CRIE SUA PRÓPRIA MODELO ULTRA‑REALISTA” e botões pra Login/Cadastro levando pro seu Clerk.
A ideia é: você só troca as imagens em /public/landing/* e ajusta textos.

function Pill({ children }: { children: React.ReactNode }) {
  return (
    <span className="inline-flex items-center rounded-full border border-neutral-200 bg-white px-3 py-1 font-semibold">
      {children}
    </span>
  );
}

function Polaroid({ className, children }: { className?: string; children: React.ReactNode }) {
  return (
    <div
      className={[
        "absolute z-10 h-64 w-72 rounded-3xl bg-white p-3 shadow-xl ring-1 ring-neutral-200",
        className ?? "",
      ].join(" ")}
    >
      <div className="relative h-[190px] w-full overflow-hidden rounded-2xl bg-neutral-100">
        {children}
      </div>
    </div>
  );
}

function Caption({ children }: { children: React.ReactNode }) {
  return (
    <div className="mt-3 text-[11px] font-semibold uppercase tracking-widest text-neutral-700">
      {children}
    </div>
  );
}

function Sticker({
  className,
  src,
  alt,
  w,
  h,
}: {
  className?: string;
  src: string;
  alt: string;
  w: number;
  h: number;
}) {
  return (
    <div className={["absolute z-20 drop-shadow-[0_18px_30px_rgba(0,0,0,0.18)]", className ?? ""].join(" ")}>
      <Image src={src} alt={alt} width={w} height={h} />
    </div>
  );
}

4) Tailwind (se você já usa)
Esse código assume Tailwind. Se você não usa Tailwind ainda, me diga que eu te mando a versão com CSS Modules (sem Tailwind).
5) Onde colocar os links de login/cadastro
Eu apontei para:
• /sign-in
• /sign-up
Que é o padrão do Clerk quando você cria rotas no App Router. Se seus caminhos forem diferentes (ex.: /auth/sign-in), troque os href.
6) Como “bater” exatamente o estilo da colagem da referência
Pra ficar idêntico (nível editorial), use este checklist:
• 3–4 “polaroids” com border + sombra e rotação leve
• 2–3 “objects/stickers” PNG com fundo transparente (camera, concha, fita, etc.)
• fundo com grid suave
• tipografia forte (headline em black), texto pequeno “magazine labels”
